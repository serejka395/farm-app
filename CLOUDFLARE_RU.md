# Очень подробная инструкция по Cloudflare

Выполняйте эти шаги по одному. Не торопитесь.

## Этап 1: Вход в настройки домена
1. Зайдите на сайт [dash.cloudflare.com](https://dash.cloudflare.com).
2. На главной странице вы увидите список ваших сайтов.
3. Нажмите на **farm2earn.space**.

---

## Этап 2: Настройка DNS (Чтобы сайт открывался)
1. В меню слева найдите пункт **DNS**. Нажмите на него.
2. Откроется подменю. Нажмите **Records**.
3. Справа вы увидите таблицу с записями ("DNS Records").
4. **Удаление старых записей (если есть):**
   - Посмотрите, есть ли записи с Type **A** и Name **farm2earn.space** (или `@`). Если есть — нажмите кнопку **Edit**, затем **Delete**.
   - Посмотрите, есть ли записи с Type **CNAME** и Name **www**. Если есть — удалите их так же.

5. **Добавление первой записи (Для самого сайта):**
   - Нажмите синюю кнопку **Add record**.
   - **Type**: Выберите **A**.
   - **Name (required)**: Напишите символ `@` (собачка).
   - **IPv4 address (required)**: Скопируйте и вставьте цифры: `76.76.21.21`
   - **Proxy status**: Убедитесь, что переключатель включен (горит оранжевым, написано **Proxied**).
   - Нажмите **Save**.

6. **Добавление второй записи (Для www):**
   - Снова нажмите **Add record**.
   - **Type**: Выберите **CNAME**.
   - **Name (required)**: Напишите `www`
   - **Target (required)**: Вставьте текст: `cname.vercel-dns.com`
   - **Proxy status**: Должно быть оранжевым (**Proxied**).
   - Нажмите **Save**.

---

## Этап 3: Настройка шифрования (Чтобы не было ошибок перезагрузки)
1. В меню слева найдите пункт **SSL/TLS**. Нажмите на него.
2. Сразу откроется страница **Overview**.
3. Справа вы увидите 4 режима (Off, Flexible, Full, Full Strict).
4. **ОБЯЗАТЕЛЬНО** выберите **Full (Strict)**. Это самый последний пункт.
   *Если выберете другой, сайт не откроется.*

5. Теперь в меню слева (в разделе SSL/TLS) нажмите на **Edge Certificates**.
6. Прокрутите вниз до пункта **Always Use HTTPS**.
7. Переключите тумблер в положение **On** (зеленый).

---

## Этап 4: Специальное правило (Чтобы сертификаты обновлялись)
1. В меню слева найдите пункт **Rules**. Нажмите на него.
2. В выпавшем списке выберите **Page Rules**.
3. Нажмите синюю кнопку **Create Page Rule**.

**Заполняем форму:**
1. **URL (required)**:
   Скопируйте эту строку и вставьте в поле:
   `*farm2earn.space/.well-known/acme-challenge/*`

2. **Pick a Setting (Choose a setting)**:
   - Нажмите на список и выберите **SSL**.
   - Справа выберите статус **Off**.

3. **+ Add a Setting** (Нажмите эту кнопку, чтобы добавить еще настройку):
   - Выберите **Cache Level**.
   - Справа выберите **Bypass**.

4. **+ Add a Setting** (Добавляем третью):
   - Выберите **Security Level**.
   - Справа выберите **Essentially Off**.

5. Нажмите синюю кнопку **Save and Deploy Page Rule**.

---

**ВСЕ!** Вы великолепны.
Закройте Cloudflare. Через 5 минут сайт `farm2earn.space` начнет работать.
